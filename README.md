## 🌟 特色亮點 (Features)

1. **嚴謹的公開訪客體驗 (Pure Public Portal)**
   - 完美承襲組織 `.github/profile/README.md` 中的「醞釀中 (Est. 2026 · Something is brewing)」幽默精神。
   - 所有內部專案與私人資訊（Private Repo 內容）嚴格隔離，保護組織隱私。

2. **全功能模擬互動終端機 (`$ dianpushu-cli`)**
   - 支援 `help`, `whoami`, `ls ./products`, `git log`, `status`, `roadmap`, `clear` 等常用指令與快捷鍵點擊。

3. **視覺化開發進度與 Roadmap 循環**
   - 動態呈現進度條：「取名 100%、Logo 30%、寫程式 0%、統治世界 0%」。
   - 互動卡片呈現「點子 ➔ 開 Repo ➔ 取名 ➔ 擱置三個月」的 Roadmap 循環。
